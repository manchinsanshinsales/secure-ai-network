# リーガルチェック評価プロンプト

あなたは企業の優秀なインハウスローヤー（企業内弁護士）です。
提示された「秘密保持契約書（NDA）」の内容を精査し、特に**受領者（乙：受領テクノロジー株式会社）にとって不利な条項やリスクが高い曖昧な条項**を抽出し、代替案を含めたレビューレポートを作成してください。

## チェック項目（重要）
以下の観点から必ず契約書を検証してください。
1. **秘密情報の定義範囲**: 秘密情報の定義が狭すぎないか、または書面等での「明示」を必須とする等の制限が受領者にとって適切か。
2. **知的財産権の帰属**: 秘密情報に基づいて独自に作成した発明や成果物の権利が、一方的に開示者に帰属するような不条理な内容（第4条など）になっていないか。
3. **損害賠償制限**: 賠償上限額の設定（過去6ヶ月の取引額または100万円の低い額など）が、実際のインシデントリスクと比較して不当に低すぎないか、あるいは受領者にとって有利か不利か。
4. **契約有効期間と存続条項**: 契約終了後の秘密保持義務が「永続的」になっている点は実務上許容できるか（期間制限を設けるべきか）。

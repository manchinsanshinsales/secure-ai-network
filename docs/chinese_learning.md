# 中国語学習ノート 🇨🇳

日常会話（Duolingo）と、中国のIT・AIトレンド情報の読解（テックニュース）を両立するための個人学習ノートです。
毎週月曜日に配信されるテックニュースと、日々の学習で学んだ単語をここに蓄積していきます。

---

## 📅 3ヶ月ロードマップ

### 1ヶ月目：発音の基礎とアプリの習慣化（7月中旬〜8月中旬）
- **目標**: Duolingoを毎日15分継続し、ピンイン（発音表記）と四声（声調）の基本ルールを覚える。
- **ニュース**: 週に1回、AI翻訳を使いながら中国語の簡体字（簡略化された漢字）の文章に触れる。

### 2ヶ月目：基本文法とIT単語のインプット（8月中旬〜9月中旬）
- **目標**: 日常会話の基本文法と、IT分野の重要単語（AI、ロボットなど）を覚える。
- **ニュース**: 週に1回、配信されるニュース記事の中から重要単語を10個ピックアップして暗記する。

### 3ヶ月目：実践的な読解と日常表現（9月中旬〜10月中旬）
- **目標**: 興味のあるITニュースのタイトルや短文を、辞書を引きながら自力で読める部分を増やす。

---

## 📈 学習のルール
1. **毎日15分**: 隙間時間にスマホでDuolingoを進める。
2. **毎週月曜日**: Antigravityが用意するテックニュースを読み、本ノートに追記する。
3. **アウトプット**: 新しく覚えた単語や気づきを、各週のセクションにメモする。

## 📰 週刊テックニュース・アーカイブ

### 【サンプル・第0回】中国のAIモデルの動向 (2026-07-15)
中国のAI技術の発展に関する短いニュースサンプルです。

#### 1. 中国語本文（簡体字）
> 中国的人工智能技术发展迅速。最近，开源模型如 Qwen 和 DeepSeek 在国际上受到广泛关注。

#### 2. ピンイン（発音表記）
> Zhōngguó de réngōng zhìnéng jìshù fāzhǎn xùnsù. Zuìjìn, kāiyuán móxíng rú Qwen hé DeepSeek zài guójì shang shòudào guǎngfàn guānzhù.

#### 3. 日本語訳
> 中国の人工知能（AI）技術は急速に発展しています。最近、QwenやDeepSeekなどのオープンソースモデルが国際的に広く注目を集めています。

#### 4. 重要単語リスト
| 簡体字 | ピンイン | 意味 |
| :--- | :--- | :--- |
| **人工智能** | réngōng zhìnéng | 人工知能（AI） |
| **技术** | jìshù | 技術 |
| **发展** | fāzhǎn | 発展する、開発する |
| **迅速** | xùnsù | 急速な、スピーディーな |
| **最近** | zuìjìn | 最近 |
| **开源** | kāiyuán | オープンソース（開発資金の源を開くことから） |
| **模型** | móxíng | モデル |
| **关注** | guānzhù | 注目する、関心を寄せる |

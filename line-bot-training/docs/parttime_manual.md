# 【アルバイト用】LINEボット開設＆スキルテスト受験マニュアル

このマニュアルは、店舗向けLINEボットの開設・設定手順と、業務開始前の「スキルテスト」の受験方法を説明したドキュメントです。

---

## 🎯 研修・昇給のルール
1. 本マニュアルに従って、LINEボットの開設と設定を行ってください。
2. 設定完了後、ご自身のLINEからボット宛に **「テスト開始」** と送信してテストを受験してください。
3. **テストで20点以上（3問中2問正解）を獲得して合格すると、マネージャーに通知され、時給アップ対象** となります。

---

## 📌 STEP 1: LINE Developers でのチャネル作成

1. [LINE Developers Console](https://developers.line.biz/) にログインします。
2. 「プロバイダー」を選択（または新規作成）し、**「 Messaging API 」** のチャネルを新規作成します。
3. チャネル設定画面から以下の2つの文字列をコピーして手元に控えます：
   - **Channel Secret**（「チャネル基本設定」タブ内にあります）
   - **Channel Access Token**（「Messaging API」タブ内で「発行」ボタンを押して取得）
4. 「LINE Official Account Manager」の応答設定で、応答モードを **「Webhook」** に変更します。

---

## 📌 STEP 2: Replit でのプログラム配置と設定

1. 指定された Replit テンプレートをフォーク（コピー）します。
2. 左メニューの **Secrets (鍵マーク)** を開き、以下の環境変数を設定します：
   - `LINE_CHANNEL_SECRET`: STEP 1で取得した Channel Secret
   - `LINE_CHANNEL_ACCESS_TOKEN`: STEP 1で取得した Channel Access Token
   - `GEMINI_API_KEY`: マネージャーから共有された Gemini API キー
3. 上部の **「Run」** ボタンを押してサーバーを起動します。
4. 表示された Webhook URL（例: `https://your-replit-name.replit.dev/webhook`）をコピーします。
5. LINE Developers の「Messaging API」設定の Webhook URL に貼り付け、**「Webhookの利用」を有効（ON）** にします。

---

## 📌 STEP 3: スキルテストの受験と合格判定

1. 作成した LINE 公式アカウントを自分のLINEで友だち追加します。
2. トーク画面で **「テスト開始」** と送信します。
3. ボットの指示に従って、まず「お名前（フルネーム）」を送信します。
4. 出題される 3 つの選択式テストに「A」「B」「C」「D」の英字で回答します。
5. 全問解答後、採点結果が自動表示されます：
   - **合格 (20点以上)**: マネージャーへ合格通知が届き、時給評価が更新されます🎉
   - **不合格 (10点以下)**: マネージャーに不合格が通知されます。マニュアルを読み直して再受験してください。

---

## ❓ 困ったときは
設定がうまくいかない場合やエラーが出る場合は、LINEのスクリーンショットを添えてマネージャーまでご連絡ください。

# GitHub 勉強ノート

このフォルダは、GitHub Skills の学習コースを日本語でまとめた学習ノートです。

## 対象コース

| コース | 概要 |
|--------|------|
| [Introduction to GitHub](./introduction-to-github/) | GitHub の基本操作（ブランチ・コミット・プルリクエスト）を学ぶ |
| [Communicate using Markdown](./communicate-using-markdown/) | Markdown を使って読みやすいドキュメントを作る |

## 学習の流れ

1. まず **Introduction to GitHub** でリポジトリの基礎を理解する
2. 次に **Communicate using Markdown** で文章の整形方法を学ぶ
